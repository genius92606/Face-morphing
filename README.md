繪圖技術設計與應用HW2-筆記
============
作業環境
===
Windows10, visual studio 2017, OpenCV4.1.0

方法說明
====

進入x64/Release
開啟HW_F94041107.exe

程式如何操作
==

1. 開啟執行檔後
2. 開始拉對應的vector
3. 注意不要將箭頭拉到招片以外(會有防呆)
4. 左右兩張照片向量的順序要一致
5. 拉完後點擊下方"Morphing"便會開始morphing
